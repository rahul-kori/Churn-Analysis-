# Customer Churn Analysis

## Project Overview
This project analyzes telecom customer churn to identify patterns and high-risk customer segments.

The goal is to help businesses reduce churn by understanding customer behavior and service usage patterns.

## Dataset
IBM Telco Customer Churn dataset containing 7043 customers and 21 features.

## Tools Used
- SQL Server
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## Key Analysis
- Overall churn rate calculation
- Churn by contract type
- Churn by tenure groups
- Churn by payment methods
- Customer segmentation based on charges and tenure

## Dashboard
The Power BI dashboard visualizes:

- Total customers
- Churn customers
- Churn rate
- Churn by contract
- Churn by tenure
- High-risk customer segments

## Key Insights
- Overall churn rate is approximately 26%
- Month-to-month contracts have highest churn rate
- Customers with tenure less than 1 year churn the most
- High monthly charges increase churn risk
- Electronic check payment method shows higher churn
